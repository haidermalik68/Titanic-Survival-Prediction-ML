# Titanic-Survival-Prediction-ML
Machine learning project predicting Titanic passenger survival using Random Forest, XGBoost, SVC and more. Built complete sklearn pipeline with feature engineering, hyperparameter tuning and achieved 0.77 Kaggle score.
Titanic Survival Prediction — Machine Learning Project
This project predicts survival of Titanic passengers using machine learning. Complete pipeline built using sklearn including data cleaning, feature engineering, encoding and multiple classification models.
Models Used:
Random Forest, XGBoost, Decision Tree, KNN, SVC, Logistic Regression, Naive Bayes
Key Steps Performed:

Complete data cleaning and missing value handling
Feature engineering including title extraction and family size
Built sklearn pipeline with ColumnTransformer
Hyperparameter tuning using GridSearchCV and RandomizedSearchCV
Compared all models and selected best performer
Submitted predictions to Kaggle competition

Libraries Used:
Python, Pandas, Numpy, Matplotlib, Seaborn, Scikit-Learn, XGBoost
Result:
Achieved 0.77 score on Kaggle public leaderboard on first submission ✅
Dataset Source: Kaggle Titanic Competition
